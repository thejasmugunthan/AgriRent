# 🌾 AgriRent – Agricultural Equipment Rental Platform

AgriRent is a web-based platform that connects farmers with agricultural equipment owners. It helps farmers rent tools and machinery easily while allowing owners to manage and earn from their equipment. The platform focuses on affordability, accessibility, and improving agricultural productivity.

--------------------------------------------------

🚜 FEATURES
- User registration and login  
- Farmer and equipment owner roles  
- Equipment listing for rent  
- Browse and search available equipment  
- Equipment booking and rental management  
- Simple and user-friendly interface  
- Secure authentication and data handling

⚙️ INSTALLATION & SETUP

1. Clone the repository  
git clone https://github.com/your-username/agrirent.git  

2. Navigate to backend folder  
cd backend  

3. Install dependencies  
npm install  

4. Database setup  
- Create a MySQL database named: agrirent  
- Import the SQL file  
- Update database credentials in backend configuration  

5. Start the server  
npm start  

6. Open frontend files in browser or run using a local server  

--------------------------------------------------

🎯 USE CASE
AgriRent reduces the cost of farming by enabling farmers to rent expensive equipment when needed and allows equipment owners to earn income from unused machinery.
